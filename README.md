## Getting Started

### Prerequisites

This project requires the following dependencies:

- **Programming Language:** Java
- **Package Manager:** Maven

### Installation

Build Peluqueria-Canina from the source and install dependencies:

1. Clone the repository:
   ```bash
   git clone https://github.com/SebassBarreto/Peluqueria-Canina
   ```

2. Navigate to the project directory:
   ```bash
   cd Peluqueria-Canina
   ```

3. Install the dependencies:

   Using [maven](https://maven.apache.org/):
   ```bash
   mvn install
   ```

### Usage

Run the project with:

Using [maven](https://maven.apache.org/):
```bash
mvn exec:java
```

### Testing

Peluqueria-Canina uses the **[test_framework]** test framework. Run the test suite with:

Using [maven](https://maven.apache.org/):
```bash
mvn test
```
